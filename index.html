Open Source

Methlang

A compiled, statically typed systems language I built in C, used in production. Parts of Empire Roleplay's backend run on it. Compiles .meth to x86-64 with an internal PE linker, no external toolchain needed. Pointers, structs, enums, generics, defer, a GC, C interop, and a standard library.
